Sobre o Mysql

É o banco de dados de código aberto mais conhecido no mundo. De acordo com o DB-Engines, o MySQL é o segundo banco de dados mais popular, ficando atrás do Oracle Database. Alimenta muitas das aplicações mais acessadas, como Facebook, Twitter, Netflix, Uber e Spotiify.
Desenvolvido e mantido pela Oracle, como é de código aberto, inclui vários recursos desenvolvidos em estreita cooperação com os usuários há mais de 25 anos. Portanto, é muito provável que sua aplicação ou linguagem de programação que utiliza seja compatível com o MySQL Database. Neste tutorial, vamos explorar um dos sistemas de gerenciamento de banco de dados relacionais mais confiavéis e amplamente utilizado.

Pré requisitos

SQL (Structured Query Language) é a forma padrão de criar, gerenciar e se comunicar com bancos de dados. É a linguagem usada para trabalhar com alguns dos bancos de dados mais populares, como PostgreSQL, SQLite e MySQL. Portanto, para seguir este tutorial, recomendamos um conhecimento básico de SQL.
